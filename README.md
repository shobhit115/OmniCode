OmniCode is a robust, all-in-one web application for generating linear barcodes (1D) and QR codes (2D). Built with Python and Streamlit, it offers real-time previews, extensive customization options, and high-resolution downloads.

## Link --> https://omnicode.streamlit.app/

## ‚ú® Features

- Dual Generation Modes:

    - Linear Barcodes (1D): Supports standard formats including EAN-13, UPC-A, Code 128, Code 39, ISBN, and more.

    - QR Codes (2D): Generate codes for URLs, text, Wi-Fi credentials, and vCards.

- Real-Time Preview: See your barcode update instantly as you type.

- Full Customization:

    - Adjust bar colors and background colors.

    - Modify dimensions (bar width, height, box size).

    - Set error correction levels for QR codes.

- High-Quality Export: Download your generated codes as high-resolution PNG files.

## üöÄ Installation & Usage

Prerequisites

Ensure you have Python 3.11 or later installed.

1. Clone the repository
```
git clone [https://github.com/shobhit115/OmniCode.git](https://github.com/shobhit115/OmniCode.git)
cd omnicode
```

2. Install dependencies

```
pip install streamlit python-barcode qrcode pillow
```

1. Run the application

```
streamlit run barcode_app.py
```
